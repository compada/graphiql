# GraphiQL

Simple html page that loads graphiql.js.

## Getting Started

This service makes heavy use of docker and docker compose. Setting up docker is beyond the scope of this readme.

```shell
cp .env.sample .env
docker compose up -d
```

## Usage

Visit <http://localhost:8080>
